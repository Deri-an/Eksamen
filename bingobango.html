Git er et verktøy som brukes for å holde styr på endringer i filer, håndtere samarbeid med andre uten problemer. For programutvikling er dette nyttig, hvor flere utviklere kan jobbe samtidig uten å skape problemer. Github er en nettbasert plattform bygget på git. Her kan utviklere lagre koden sin, jobbe sammen og bidra til bedre prosjekter med funksjoner som pull og requests og oppgavelister. Disse funksjonene gjør samarbeidet lettere. Sammen hjelper Git og Github enkltpersoner og team med å organisere, utvikle og vedlikeholde programvare på en smart og effektiv måte. Repositorier på Github har gjort det mye lettere å jobbe sammen for utviklingsteam, uansett hvor de befinner seg. Github fungerer som plattform hvor man kan lagre, dele og holde orden på koden. Dette fører til at verktøy blir sett på som viktig i moderne programutvikling. Versjonskontroll: Github bruker Git og lar deg spore endringer i koden, altså gå tilbake til tidligere versjoner og holde oversikt over alle endringer i prosjektet. Samarbeidsfunksjoner: Funksjoner som pull requests og kodegjennomgang, kan utviklere enkelt foreslå, diskutere og legge inn endringer på en enkel måte. Dette gir en god oversikt og sikrer høy kvalitet av koden. Fellesskap og åpen kildekode: Github har mange åpne kildekodeprosjekter, der utviklere kan bidra, lære av andre og bruke felles ressurser for å jobbe mer effektivt/bruke andre arbeid for å jobbe raskere.
